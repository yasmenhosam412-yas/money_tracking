# imrpo 💰

A modern Flutter personal finance application for tracking income, expenses, and savings goals with a clean and intuitive user experience.

## ✨ Features

- Income and expense management
- Savings goals and financial planning
- Balance dashboard and analytics
- Multi-currency support
- English & Arabic localization with RTL support
- Secure authentication system
- Password reset and profile management
- Account deletion support
- Responsive and modern UI

## 🏗️ Architecture

The project follows **Clean Architecture** principles with:

- **BLoC** for state management
- **GetIt** for dependency injection
- Repository pattern
- Feature-based scalable structure

## ☁️ Backend & Database

- Firebase Authentication & Firestore
- Supabase Authentication & PostgreSQL

## 🛠️ Tech Stack

- Flutter
- Dart
- BLoC
- GetIt
- Supabase
- Firebase
- PostgreSQL
